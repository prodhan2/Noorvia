### 📖 ইলম (জ্ঞান)

#### কুরআন শরীফ
- ১১৪টি সূরার তালিকা — বাংলা নাম, আরবি নাম, অর্থ, আয়াত সংখ্যা, মাক্কী/মাদানী ব্যাজ
- সূরা নম্বর বা নাম দিয়ে সার্চ করার সুবিধা
- সূরা, পারা (৩০ পারা) এবং পৃষ্ঠা (৬০৪ পৃষ্ঠা) — তিনটি ট্যাবে নেভিগেশন
- সূরার বিস্তারিত পেজ — আয়াত পড়া ও তিলাওয়াত

- বাংলা অনুবাদসহ কুরআন পড়ার সুবিধা

- ফ্লোটিং অডিও প্লেয়ার — যেকোনো স্ক্রিনে থেকেও চলতে থাকে

#### হাদীস
- হাদীস পড়ার সুবিধা (ডেমো পেজ)

#### নামায শিক্ষা (নামাযের নিয়ম)
- নামাযের নিয়ম ও পদ্ধতি বিস্তারিতভাবে

#### রুকইয়াহ
- রুকইয়াহর তালিকা ও বিস্তারিত পেজ

#### আসমাউল হুসনা
- আল্লাহর ৯৯টি নাম ও অর্থ

#### আরবি বর্ণমালা শিক্ষা
- ২৮টি আরবি অক্ষর — বাংলা নাম, উচ্চারণ, অডিও
- প্রতিটি অক্ষরের বিস্তারিত পেজ — লেখার ধরন, উদাহরণ
- অনুশীলন পেজ

- প্রগ্রেস ট্র্যাকিং — কতটি অক্ষর শেখা হয়েছে
- স্ট্রিক কাউন্টার — ধারাবাহিক অনুশীলনের রেকর্ড
- ফেভারিট অক্ষর সেভ করার সুবিধা
- সার্চ — অক্ষর, বাংলা বা ইংরেজি নাম দিয়ে খোঁজা

#### ইসলামিক নাম

- নামের অর্থ ও বিস্তারিত তথ্য

- ১০টি জনপ্রিয় যিকির — সুবহানাল্লাহ, আলহামদুলিল্লাহ, আল্লাহু আকবার, লা ইলাহা ইল্লাল্লাহ, আস্তাগফিরুল্লাহ, দরূদ শরীফ, বিসমিল্লাহ, ইয়া আল্লাহ, সুবহানাল্লাহি ওয়া বিহামদিহি, লা হাওলা ওয়ালা কুওয়াতা
- কাস্টম লক্ষ্যমাত্রা নির্ধারণ
- প্রগ্রেস বার ও পরিসংখ্যান
- লক্ষ্য পূরণে সাফল্যের বার্তা
- ডেটা সেভ — অ্যাপ বন্ধ করলেও কাউন্ট থাকে

- **ডার্ক মোড / লাইট মোড** — ম্যানুয়াল বা অটো (সকাল ৬টা–সন্ধ্যা ৬টা)

## 🚀 ইনস্টলেশন ও রান

### প্রয়োজনীয়তা
- Flutter SDK (^3.9.2)
- Dart SDK
- Android Studio / VS Code
- Firebase প্রজেক্ট (Android ও iOS এর জন্য)

### ধাপসমূহ

```bash
# ১. রিপোজিটরি ক্লোন করুন
git clone https://github.com/your-username/noorvia.git
cd noorvia

# ২. ডিপেন্ডেন্সি ইনস্টল করুন
flutter pub get

# ৩. অ্যাপ রান করুন
flutter run
```

### Firebase সেটআপ
১. [Firebase Console](https://console.firebase.google.com) এ নতুন প্রজেক্ট তৈরি করুন
২. Android ও iOS অ্যাপ যোগ করুন
৩. `google-services.json` (Android) এবং `GoogleService-Info.plist` (iOS) ডাউনলোড করে সঠিক ফোল্ডারে রাখুন
৪. Firebase Authentication ও Firestore চালু করুন

### বিল্ড

```bash
# Android APK
flutter build apk --release

# Android App Bundle
flutter build appbundle --release

# iOS
flutter build ios --release

# Web
flutter build web --release
```
